Welcome to Production Buddy

      

      

        
Production Buddy uses a small local AI model (Ollama) to interpret less-obvious PDF markup — things like "add a check mark to the row above" or "delete this print ads line". Everything runs on your machine — no data leaves your computer, no subscription, no API keys.

        
It's a one-time ~5 GB download. You can also skip this and the app will still handle the most common edits using built-in pattern matching.


        

          

            
            
Opens the official Ollama download page in your browser. After installing, return here and the app will detect it automatically.

          

          

            
            
The app will use rule-based matching only. Common edits (replace text, sort table, add row) still work; ambiguous notes will land in "Human Review" for you to handle. You can install Ollama later from the Settings panel.
